`Alt + Shift + ↑/↓`: 复制整行到下一行(光标停留在上/下行)  
`Alt + ↑/↓`: 整行向上/下移动  
`Alt + z`: 自动换行Ω  
`Alt + Shift + f`: 代码格式化  

`Command + d`: 批量修改  
`Command + Shift + k`: 删除整行  
`Command + Enter`: 在当前行下插入新的一行  
`Command + Shift + Enter`: 在当前行上插入新的一行  
`Command + Shift + 空格`: 参数提示 trigger parameter hints  
`Command + ]/[`: 行缩进  
`Command + 1/2/3`: 拆分编辑器  

`ctrl + space`: 触发提示trigger suggest 修改为`ctrl + Enter`
